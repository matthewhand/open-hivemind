## 2024-06-19 | [Architectural Audit] | Insight: Divs acting as interactive triggers (Dropdowns, Notification bells) cause severe keyboard trapping and missing accessible states. Global 'saving' booleans in data-dense list views create race conditions when editing adjacent items. | Protocol: Always use semantic <button type="button"> for interactive triggers with Tailwind focus-visible classes. Always map loading indicators to deterministic keys (e.g., savingKey) instead of global flags.
